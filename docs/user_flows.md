# User Flows

Constellation is built around a single rhythm:
Reflect → Submit → Gather → Grow.

Below are the core user flows that support this ritual.

---

# 1. Join a Constellation

graph TD
  A[Invite Link] --> B[Landing Page]
  B --> C[Enter Name + Email]
  C --> D[Create Account]
  D --> E[Assigned to Group]
  E --> F[Welcome Email Sent]
  F --> G[Await First Prompts]

Narrative:
A soft entry. The user should feel like they are joining something small, meaningful, and intentional.

---

# 2. Receive Weekly Prompts

graph TD
  A[Start of Assigned Week] --> B[Prompts Email]
  B --> C[User Opens App]
  C --> D[4 Questions Delivered]
  D --> E[Begin Reflection]

Narrative:
The questions should feel like an invitation, not a task.

---

# 3. Submit Reflections

graph TD
  A[Open Prompts] --> B[Write Reflection]
  B --> C[Add Photo / Music]
  C --> D[Set Mood Slider]
  D --> E[Save Draft]
  E --> F[Submit All]

Narrative:
From writing sanctuary → submission moment → quiet satisfaction.

---

# 4. Monthly Wrapped Compilation

graph TD
  A[Last Day of Month] --> B[Collect All Responses]
  B --> C[Analyze Themes]
  C --> D[Select Highlights]
  D --> E[Assemble Photos + Song]
  E --> F[Generate Narrative]
  F --> G[Publish Wrapped]

Narrative:
A story emerging from many voices.

---

# 5. View Monthly Wrapped

graph TD
  A[Email Notification] --> B[Open Wrapped]
  B --> C[Scroll Experience]
  C --> D[View Highlights]
  D --> E[View Group Mood]
  E --> F[Share with Group]
  F --> G[Look Ahead to Next Month]

Narrative:
A cinematic pause. A moment to breathe and witness.

---

# 6. Personal Insights Flow

graph TD
  A[User Visits Insights] --> B[View Mood Graph]
  B --> C[View Themes]
  C --> D[See Photos in Time]
  D --> E[Read Own Story Highlights]

Narrative:
A quiet mirror where a person sees the arc of their own becoming.
