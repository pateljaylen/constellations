## Steps to Sync with GitHub

### Option 1: If you already have a GitHub repository

1. **Add your GitHub remote:**
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   # OR if using SSH:
   git remote add origin git@github.com:YOUR_USERNAME/YOUR_REPO_NAME.git
   ```

2. **Stage all your changes:**
   ```bash
   git add .
   ```

3. **Commit your changes:**
   ```bash
   git commit -m "Add Constellation app: auth, groups, reflections, and documentation"
   ```

4. **Push to GitHub:**
   ```bash
   git push -u origin main
   ```

### Option 2: If you need to create a new GitHub repository

1. **Create a new repository on GitHub:**
   - Go to https://github.com/new
   - Name it (e.g., "constellation")
   - Don't initialize with README, .gitignore, or license (you already have these)

2. **Follow Option 1 steps above** using your new repo URL

### Option 3: If you want to check what's different first

1. **See what files are new/modified:**
   ```bash
   git status
   ```

2. **See what would be committed:**
   ```bash
   git add -n .  # Dry run, shows what would be added
   ```

3. **Review changes in modified files:**
   ```bash
   git diff README.md
   ```
